# 🌍 ZENTRIP – Travel Booking App UI Design

**ZENTRIP** is a visually rich and intuitive mobile UI design built for travel planning, hotel booking, food ordering, and trip management. Created using **Figma**, this project showcases a seamless and engaging travel experience — from exploration to payment.

## 🧭 Project Overview

ZENTRIP brings together various features under one roof:
- Hotel booking
- Food ordering
- Trip planning
- Messaging
- Seamless payment flow

The UI is designed with vibrant visuals, gradient accents, and a modern aesthetic to offer an immersive user experience.

## 📱 App Screens

This Figma project includes:

- Splash & Intro Screens  
- Sign-In/Sign-Up UI  
- Inspirational Trip Quotes Screens  
- Hotel Listings and Details  
- Food Menu and Ordering Flow  
- Destination Discovery  
- Booking Summary  
- Messages & Chat Interface  
- Profile & Settings Screen  
- Payment Integration UI  
- Booking Confirmation Screen  
- Tab Navigation (Trips, Hotels, Food, Bookings)

## 🔥 Features

- 📍 Hotel details with booking durations (1-day, 7-day)
- 🍲 Food menu with price and Add to Cart flow
- 🏞️ Popular destinations (Bora Bora, London, Mount Everest)
- 💬 Integrated chat with fellow travelers or agents
- 💳 Payment gateway with credit card input
- ✅ Order confirmation with visual feedback
- 🌐 Tab-based bottom navigation system

## 🛠 Tools Used

- **Figma** – UI/UX Design  
- **Custom Illustrations** – Travel icons & illustrations  
- **Google Fonts** – Clean and readable typography  
- **High-resolution mockups** – Realistic UI rendering
